# Secure JWT API

A secure Flask authentication API using JWT and bcrypt.

## Features
- User registration
- Secure password hashing
- JWT authentication
- Protected routes
- SQLite database

## Technologies Used
- Python
- Flask
- SQLite3
- PyJWT
- bcrypt

## Installation

pip install flask pyjwt bcrypt

## Run

python app.py

## API Endpoints

POST /register  
POST /login  
GET /profile

## Purpose
focuses on secure authentication systems and password hashing.
